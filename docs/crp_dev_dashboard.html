Princípio: Zero custo durante o desenvolvimento. Todo investimento ocorre apenas quando o app estiver pronto para gerar receita.

→ Primeiro gasto: Só quando o primeiro curso estiver gravado e pronto para venda. Estimativa: upgrade Supabase ($25/mês) + domínio (R$40/ano) + Google Play ($25 única).
